htmlファイルはそのままファイルから開いてください。
hexファイルはMicrobit Python Editorでのみ中身を確認、編集、Micro:bitに書き込みできます。そのまま開いても文字の羅列が並んでいるだけです。
txtファイルは、hexファイルの内容が書いてあるだけです。

HTML files: Open directly from the file.
HEX files: Can only be viewed, edited, and written to the Micro:bit using the Micro:bit Python Editor. Opening them directly will only display a sequence of characters.
TXT files: Contains the same content as the HEX files.
